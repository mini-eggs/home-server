# Home Server
This is my home server I run off a Raspberry Pi 3.

#### What this includes
A MySQL server.
A PHPMyadmin server (that will run off localhost:8080/phpmyadmin/).
A Go server (that will run off localhost:5000).

#### Requirements
Have both Docker and Go installed.

#### Setup
`$ make setup`

#### Production
`$ make`

#### Development
`$ make dev`
