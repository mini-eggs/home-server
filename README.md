# Home Server
This is my home server I run off a Raspberry Pi 3.

#### What this includes
1. A MySQL server.

2. A PHPMyadmin server (that will run off localhost:8080/phpmyadmin/).

3. A Go server (that will run off localhost:5000).

#### Requirements
Have both Docker and Go installed.

#### Setup
`$ make setup`

#### Production
`$ make`

#### Development
`$ make dev`
